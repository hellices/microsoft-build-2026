## How to read

각 세션 노트는 다음 순서로 구성됩니다.

1. **TL;DR** — 한 줄 요약
2. **Why it matters** — 고객 관점 시사점
3. **Key announcements** — GA / Preview / SDK
4. **Session summary** — 섹션별 요약
5. **Demo highlights** — 데모 타임스탬프
6. **Customer takeaways** — 도입 시 점검 항목
7. **Resources** — 원본 링크
